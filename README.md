# TCR Descomplicado: Ferramenta web para resolução de sistemas de congruências lineares

Projeto desenvolvido para a disciplina de Fundamentos de Matemática para Ciência da Computação 2 (FMCC2), ministrada por Tiago Massoni.

## 1. Objetivo

Esta ferramenta web tem como objetivo demonstrar a resolução passo a passo para a resolução de um sistema de até 5 congruências lineares utilizando o Teorema Chinês Do Resto (TCR).

## 2. Ferramentas utilizadas

Para este app, foram utilizadas as seguintes tecnologias:

1. Back-end com python/flask;
2. Front-end com html, css e javascript.

## 3. Como utilizar?

É possível entrar no site pela web entrando no [link](https://tcr-descomplicado.vercel.app/), hospedado no Vercel. Da mesma forma, é possível baixar este repositório (ou cloná-lo para sua máquina) e executar o arquivo run.py, escrevendo `python3 run.py` ou `python3 run.py` em um terminal aberto no diretório raiz do projeto.
